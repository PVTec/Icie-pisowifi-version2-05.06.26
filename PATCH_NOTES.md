Maligayang pagbabalik sa Icie Wifi Portal! Ang bersyon 1.8, na tinatawag naming "Pakikipag-ugnayan" Update, ay nakatuon sa pagpapabuti ng iyong karanasan sa pamamagitan ng paggawa ng app na mas user-friendly, mas madaling maunawaan, at mas rewarding para sa aming mga lokal na user. Isinalin namin ang malaking bahagi ng app sa Taglish at nagdagdag kami ng mga bagong feature para mas maging masaya ang iyong paggamit!

---

## ⚙️ Mga Adjustment at Pagpapabuti

### **Taglish Localization**
- **Buong App, Isang Wika:** Ang buong application—mula sa homepage, dashboard, arena, hanggang sa iyong profile—ay nasa Taglish na para mas madaling maintindihan at gamitin.
- **Lokal na Pakiramdam:** Ang mga laro, button, at mga notification ay mas pamilyar na ngayon, na nagbibigay ng mas komportableng karanasan.

### **Mas Pinadaling Sign-Up at Login**
- **Username Validation:** Ang username field sa sign-up at login pages ay mamumula na ngayon at magbibigay ng babala kung ang user ay magta-type ng mga bawal na character tulad ng '@' o space.
- **Mas Malinaw na UI:** Tinanggal ang mga nakakalitong UI elements para mas diretso sa punto ang paggawa ng account at pag-log in.
- **Sign-Up Success Modal:** Pagkatapos gumawa ng account, isang pop-up message na ngayon ang lilitaw para ipaalam sa user ang kanilang bagong username at gabayan sila sa pag-login.

---

## 🐞 Mga Inayos na Bug

- **General Stability:** Nagpatupad kami ng mga pag-aayos sa backend at sa database security rules para maiwasan ang mga "missing or insufficient permissions" error, lalo na sa panahon ng pag-sign up at pagbili ng voucher.
- **UI Fixes:** Inayos ang ilang maliliit na isyu sa layout at display sa iba't ibang bahagi ng app para sa mas malinis na itsura.
- **Performance:** Pinabuting ang data fetching sa ilang mga pahina para mas mabilis ang loading time.

# Icie Wifi Portal: Overview & Features

## Latest Updates (Patch v1.7 - The Investment Update)

This major update introduces a high-stakes **Credit Investment** feature, completely overhauling the portal's economy and adding a new layer of strategy for users.

-   **New Feature: The Bank & Wallet System**:
    -   **Wallet**: A personal savings area where users can cash in credits from their main balance.
    -   **Bank**: The high-risk investment portfolio. The Bank is **locked** until a user makes their first voucher purchase, incentivizing engagement with the shop.

-   **High-Risk, High-Reward Investment Gameplay**:
    -   Users can invest credits from their wallet (up to a 5,000 credit limit) into the Bank.
    -   The investment runs automatically, with a **35% chance to win** and a **65% chance to lose** on each "tick."
    -   Gains and losses are calculated based on a percentage of the **initial investment amount**, offering massive potential returns.
    *   A "Growth Expectation" is now clearly displayed in the investment hub, showing users their potential target.

-   **Strategic Depth & New Rules**:
    -   **Time-Locked Durations**: Investments are locked for a minimum of 3 minutes. Users can set a custom duration, but cannot stop the investment prematurely.
    -   **Disconnection Penalty**: To ensure fairness, if a user logs out or disconnects while an investment is active, **45% of their Bank balance is automatically deducted**.
    -   **Withdrawal Fees**: A **15% fee** is now applied when withdrawing from the Bank to the Wallet.
    -   **Automatic Stop**: The investment automatically stops if the Bank balance hits zero or the chosen duration ends.

-   **New User Incentives & Onboarding**:
    -   **Investment Seed Fund**: First-time visitors to the invest page receive a **500 credit bonus** to their Wallet.
    -   **Guided Feature Unlocking**: The Bank feature and investment modals only appear after a user's first purchase, creating a guided experience.
    -   **Low Credit Notifications**: A modal now prompts users to try the investment feature if their main credit balance drops below 10.
    -   **Notification System**: A dot appears on the "Invest" navigation link to notify users of a finished investment or if they've never invested before.

-   **Bug Fixes & UI Polish**:
    -   Fixed multiple UI crashes related to missing component imports in modals.
    -   Redesigned and cleaned up the "Investment Hub" modal for a more comfortable and intuitive user experience.

# Icie Wifi Portal - Patch Notes 1.6

This update introduces major social and administrative features, focusing on user connectivity and advanced session management. We've also pushed numerous UI/UX enhancements to create a cleaner, more comfortable experience across the entire application.

## What's New?

### 1. Online Users Hub & Enhanced Socials
- **See Who's Online**: A new "Online Users" page is now available from the Arena, showing all players active in the last 5 minutes.
- **Direct Interaction**: From the Online Users list, you can now send a private chat message or initiate a Tic-Tac-Toe battle challenge with a single click.
- **Live Notifications**: The app now features a real-time notification system.
    - **Incoming Messages**: Get an instant pop-up modal for new private messages when you're not in a chat window, so you never miss a conversation.
    - **Battle Challenges**: Receive modals for new Tic-Tac-Toe challenges and when an opponent accepts your challenge.

### 2. Advanced Admin Device & Access Management
The "Vench" admin panel has been upgraded with a powerful **Device Management** tab, giving you complete control over user access.
- **Categorized User Lists**: Instantly see which users are:
    - Actively using a Universal Access pass.
    - Currently online and connected to the "Icie Wifi" network.
    - Inactive or offline.
- **Revoke Access**: Immediately disconnect a user with Universal Access. The user is instantly notified of the change.

### 3. Universal Access Request System
- **User Request Page**: Users who can't connect to the local WiFi can now navigate to a dedicated page to send an official request for Universal Access.
- **Admin Approval Workflow**: A new "Access Requests" section appears in the Vench panel, allowing the admin to **Grant** or **Deny** requests with one click.
- **Automated Notifications**: Users are automatically notified via a toast message whether their request was approved or denied.

## Improvements & Fixes

- **UI/UX Overhaul**:
    - **More Comfortable Layout**: Enhanced spacing and card widths on the Dashboard and Profile pages for a less "slim" and more breathable design.
    - **Cleaner Arena**: The Tic-Tac-Toe game screen has been refined for better visual clarity and responsiveness on all devices.
    - **Homepage Polish**: Improved the structure and spacing of the landing page for a more professional and inviting first impression.
- **Bug Fixes**:
    - Fixed an issue where text would overflow on Theme/Effect shop cards.
    - Corrected button alignment and icon spacing for a cleaner look.
    - Resolved a client-side error (`getDoc is not defined`) in the notification system.
    - The new message notification modal is now intelligently disabled when the user is already in a chat room, preventing disruptive and redundant pop-ups.
- **New Access Rules**: Universal Access is now a temporary 36-hour pass, creating a more engaging incentive system. This time-limited access correctly governs entry to the Arena, Inbox, and all Chat features.
